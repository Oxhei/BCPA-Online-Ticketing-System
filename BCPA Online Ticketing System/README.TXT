Project Title:
BCPA Online Ticket System

Purpose of Project:
This system enables managers to manage event information, including prices and names. It also allows customers to purchase tickets for shows.

Version:
0.2

Authors:
Richard and Brunaldo

User Instructions
1. To begin, create an object from the Manager class. Right-click on the Manager class and select 'new Manager()'.
2. A dialog box will appear. Click 'OK' to proceed.
3. This action creates a red box at the bottom of the screen.
4. Right-clicking on this red box will display a list of available methods for user interaction.
